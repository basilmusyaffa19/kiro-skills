# Kiro Skills

A collection of reusable [Kiro](https://kiro.dev) skills for AI-assisted development workflows.

## Available Skills

| Skill | Description |
|-------|-------------|
| [technical-document-writer](skills/technical-document-writer/) | Write internal technical documents as numbered Obsidian Markdown files and convert them to styled Word documents (.docx). |

## Installation

Copy the desired skill folder into your project's `.kiro/skills/` directory:

```bash
# Example: install technical-document-writer
cp -r skills/technical-document-writer /path/to/your-project/.kiro/skills/
```
